# 🎮 Catch the Falling Objects

A fun browser game built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## Features

- 🎯 Catch falling objects
- ❤️ Lives system
- 📈 Increasing difficulty
- 🏆 Score tracking
- ⭐ Level progression
- 📱 Responsive design
- ✨ Smooth animations
- 🎨 Modern Glassmorphism UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

1. Download or clone the repository.
2. Open `(https://kalashsh.github.io/Catch-the-Falling-Objects/)` in your browser.

## Controls

- ⬅ Left Arrow
- ➡ Right Arrow

## Future Improvements

- Power-ups
